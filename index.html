Concert Archive
Setlist database with photos and videos.
Search through shows, and view or upload photos and videos from each date. View song statistics including set openers, closers, encores, and percentage played.
ReactJS front end with a Rails back end. Data is stored in a PostgreSQL relational database. Integrated Cloudinary for media storage and transformations. Styling with Semantic-UI React.
Audio Advocate
Compare ratings and reviews for audio gear.
The app uses data gathered from pro audio retailers including Amazon, Sweetwater, and Guitar Center. Sort products based on average rating, number of ratings, and price, or filter by category. Product view features individual customer reviews and ratings.
Product card and review components were created using a ReactJS/Redux front end. The back end is a Rails API which handles the sorting and filtering of products based on different criteria. Data was gathered by scraping music retail websites using Nokogiri. All data is stored in a PostgreSQL database. Constellation Playground
Create your own constellations.
Compose, name, and save your own constellations in a given set of stars. Check out the Constellation Community to view other users' skies.
Interactive star and line components were created with a ReactJS/Redux front end. The user experience (UI/UX) consists of Scalable Vector Graphics (SVG) that respond to mouse hovers and clicks. Constellation and star data is stored in a PostgreSQL database with logic built on a Rails API. A secure user sign-up and login experience was constructed with JWT authorization.
Battle of Wits
Battle your friends in a trivia contest.
Play a 2-player trivia game with audio and visual feedback.
Question and answer data is gathered by making Ajax calls to Open Trivia DB. jQuery is used to display questions and set a response timer. Audio clips are triggered to provide users with feedback upon response submission. Uses the Materialize CSS framework for styling.
